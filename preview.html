ManiX Wallpaper Preview

Use this tool to preview wallpapers and generate JSON entries for your wallpapers.json file.

External URL Wallpaper

For external URLs (like Unsplash, Imgur, etc.), you need to provide both the main wallpaper URL and a thumbnail URL.

Local Upload Wallpaper

For GitHub-hosted images:

1. Upload your image to the images folder in this repository

2. For the thumbnail, either:

  • Upload a smaller version of the image with -thumbnail suffix (e.g., ocean-thumbnail.jpg)
  • Or use the same image URL for both main image and thumbnail

3. The raw GitHub URL format will be:

https://raw.githubusercontent.com/YOUR_USERNAME/ManiX_Wallpapers_pack/main/images/your-image.jpg
